# ArgoCD GitOps Deployment Lab

This repository documents the implementation of **GitOps** principles using **ArgoCD** to automate application deployments on a Kubernetes cluster.

## 📋 Project Overview
The lab focuses on automating the deployment process with the following key configurations:
- **Project Structure**: Targeting a specific Kubernetes namespace (`hamdy`).
- **Automation**: Enabled **Auto-Sync** and **Self-Healing** to ensure the cluster state always matches the desired state in this Git repository.
- **Resources**: Includes Deployment, Service, and ConfigMap definitions.
